Determine if note or args is a rest

rest?

[note_or_args (number_symbol_or_map)]

Given a note or an args map, returns true if it represents a rest and false if otherwise

Introduced in v2.1 

Example 0 

puts rest? nil




# true





Example 1 

puts rest? :r




# true





Example 2 

puts rest? :rest




# true





Example 3 

puts rest? 60




# false





Example 4 

puts rest? {}




# false





Example 5 

puts rest? {note: :rest}




# true





Example 6 

puts rest? {note: nil}




# true





Example 7 

puts rest? {note: 50}




# false
