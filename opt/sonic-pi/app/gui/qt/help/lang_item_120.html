Enable and disable arg checks

use_arg_checks

[true_or_false (boolean)]

When triggering synths, each argument is checked to see if it is sensible. When argument checking is enabled and an argument isn't sensible, you'll see an error in the debug pane. This setting allows you to explicitly enable and disable the checking mechanism. See with_arg_checks for enabling/sisabling argument checking only for a specific do/end block.

Introduced in v2.0.0 

Example 0 


play 50, release: 5
use_arg_checks false
play 50, release: 5




 
 # Args are checked
 
 # Args are not checked
