Construct chords based on scale degrees

chord_degree

[degree (symbol_or_number), tonic (symbol), scale (symbol), number_of_notes (number)]

A helper method that returns a list of midi note numbers when given a degree (a symbol :i,:ii,:iii,:iv,:v :vi, :vii or a number 1-7), tonic, scale and number of notes

Introduced in v2.1 

Example 0 

puts chord_degree(:i, :A3, :major)




# returns a list of midi notes - [69 73 76 80]





Example 1 

play chord_degree(:i, :A3, :major)
