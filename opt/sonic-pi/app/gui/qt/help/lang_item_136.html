Use sample pack

use_sample_pack

[pack_path (string)]

Given a path to a folder of samples on your filesystem, this method makes any wav|wave|aif|aiff files in that folder available as samples. Consider using use_sample_pack_as when using multiple sample packs.

Introduced in v2.0.0 

Example 0 


use_sample_pack '/home/yourname/path/to/sample/dir'
sample :foo 




 
 
 #=> plays /home/yourname/path/to/sample/dir/foo.{wav|wave|aif|aiff}
