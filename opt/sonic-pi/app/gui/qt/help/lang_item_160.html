Create chord

chord

[tonic (symbol), name (symbol)]

A helper method that returns a list of midi note numbers when given a tonic note and a chord type

Introduced in v2.0.0 

Example 0 


puts chord(:e, :minor)




 
 # returns a list of midi notes - [64, 67, 71]





Example 1 


# Play all the notes together
play chord(:e, :minor)



 
 
 





Example 2 


# looping over arpeggios can sound good
# Here we use choose to pick a random note from the chord
loop do
  play chord(:e, :minor).choose
  sleep 0.2
end
