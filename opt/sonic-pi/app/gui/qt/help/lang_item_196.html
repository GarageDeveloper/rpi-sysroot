Create scale

scale

[tonic (symbol), name (symbol)]

A helper method that returns an Array of midi note numbers when given a tonic note and a scale type. Also takes an optional :num_octaves parameter (1 octave is the default)

Introduced in v2.0 

Example 0 

puts scale(:C, :major)




# returns the list [60, 62, 64, 65, 67, 69, 71, 72]





Example 1 


play_pattern scale(:C, :major)



# anywhere you can use a list of notes, you can also use scale
 





Example 2 


play_pattern(:C, :major, num_octaves: 2)



# you can use the :num_octaves parameter to get more notes
