Use_random_seed

use_random_seed

[seed (number)]

Resets the random number generator to the specified seed. All subsequently generated random numbers will use this new generator and the current generator is discarded. Use this to change the sequence of random numbers in your piece in a way that can be reproduced

Introduced in v2.0.0 

Example 0 


use_random_seed 1
puts rand
use_random_seed 1
puts rand 




 
 # reset random seed to 1
 # => 0.417022004702574
 # reset random seed back to 1
 #=> 0.417022004702574
