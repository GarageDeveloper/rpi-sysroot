Get note info

note_info

[note (symbol_or_number)]

Returns an instance of SonicPi::Note. Please note - :octave param is overridden if octave is specified in a symbol i.e. :c3

Introduced in v2.0.0 

Example 0 


puts note_info(:C, octave: 2)
# returns




 
 
 #<SonicPi::Note:0x0000010206bf78 @pitch_class="C", @octave=2, @interval=0, @midi_note=36, @midi_string="C0">
