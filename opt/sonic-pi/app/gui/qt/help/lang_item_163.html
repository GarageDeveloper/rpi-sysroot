Knit a sequence of repeated values

knit

[start (number), finish (number), step_size (number)]

Knits a series of value, count pairs to create a ring buffer where each value is repeated count times.

Introduced in v2.2 

Example 0 

(knit 1, 5)   




#=> (ring 1, 1, 1, 1, 1)





Example 1 

(knit :e2, 2, :c2, 3)




#=> (ring :e2, :e2, :c2, :c2, :c2)
