Pre-load sample

load_sample

[path (string)]

Given a path to a wav|wave|aif|aiff file, this loads the file and makes it available as a sample. See load_samples for loading multiple samples in one go.

Introduced in v2.0.0 

Example 0 


load_sample :elec_blip
sample :elec_blip




 
 # :elec_blip is now loaded and ready to play as a sample
 # No delay takes place when attempting to trigger it
